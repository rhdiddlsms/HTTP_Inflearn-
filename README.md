# HTTP

실무에서 필요한 HTTP 핵심 기능과 올바른 HTTP API 설계 방법 학습하기.

- 네트워크 기본
- HTTP 핵심 이해
- HTTP API 설계 방법

## 목표

- HTTP의 전체 흐름 이해
- 실무에서 꼭 필요한 핵심내용 정리
- 수 많은 예시와 그림으로 설명

## 대상

- 모바일 앱
- 웹 프론트엔드
- 백엔드

## 강의 순서

1. 인터넷네트워크
2. URI와 웹브라우저 요청 흐름
3. HTTP 기본
4. HTTP 메서드
5. HTTP 메서드 활용
6. HTTP 상태코드
7. HTTP 헤더1_일반헤더
8. HTTP 헤더2_캐시와 조건부 요청
